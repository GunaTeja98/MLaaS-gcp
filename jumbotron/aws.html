AWS offers a broad set of machine learning services with supporting cloud infrastructure, putting machine learning in the hands of every developer, data scientist, and expert practitioner.



What is AWS Machine Learning?

Amazon Machine Learning is a service that allows users to develop predictive applications by using algorithms, mathematical models based on the user’s data.

It provides visualization tools to guide the users through the process of creating machine learning (ML) models without having to learn complex ML algorithms and technology. It reads data through Amazon S3, Redshift, and RDS then visualizes the data through the AWS Management Console and the Amazon Machine Learning API.

This data can be imported or exported to other AWS services via S3 buckets.

It uses an “industry-standard logistic regression” algorithm to generate models.

Types of Tasks Performed by Amazon Machine Learning

Three different types of tasks can be performed by Amazon Machine learning service

  • A binary classification model: It can predict one of the two possible results, i.e. either yes or no.
  • A multi-class classification model: It can predict multiple conditions. For example, it can track customer's online orders.
  • A regression model: Regression models can predict the best-selling price for a product or the number of units that will sell.

Machine Learning Services in AWS


Amazon SageMaker is a fully managed service that can help machine learning developers and data scientists quickly build, train, and deploy models. Easy. AmazonSageMaker Autopilot can automatically check raw data, select the best set of algorithm parameters, train and tune multiple models, track model performance, and rank models based on performance. In this way, you can greatly change the time required to find the best model. Supported frameworks include: TensorFlow, PyTorch, Apache MXNet, Chainer, Keras, Gluon, Horovod, Scikit-learn and Deep Graph Library.

Amazon Comprehend service uses natural language processing (NLP) to analyze text. Comprehend allows developers to add custom lists without the need for an in-depth machine learning or natural language processing background. It will take on the responsibility of building, training and hosting customized machine learning models and provide these models through a private API

Amazon Translate is a reasonable neural machine translation service that uses advanced machine learning technology to provide rapid language translation of text-based content and supports the development of multilingual user experience applications. Enterprises can now more easily expand products and services in other regions by allowing users to use automated language translation to access websites, information, and resources in their preferred language.

Amazon Rekognition allows you to easily add image and video analysis capabilities to your application. You only need to provide images or videos to the Rekognition API, and the service can identify objects, people, text, scenes, and activities, and detect any inappropriate content. Amazon Rekognition also provides highly accurate facial analysis and facial recognition functions for the images and videos you provide.
