The training of an Machine learning model requires an ML algorithm to learn from the training data sets. By the help of ML algorithm, ML model is trained on the training datasets.

In machine learning there are several types of training:

1.) Training with IaaS:

Training the machine learning models through Infrastructure as a Service (IaaS), the cloud provider, handles the machines, their virtualization and the required networking. So, IaaS includes storage, networking, virtualization, and provides servers. The user is responsible for everything else—OS, middleware, data, and application software. The provider is the host of the resources on which the user builds the infrastructure. A user can get predictions on a new data by using this ML model. For example, checking whether a newly received email is spam or not. Google compute Engine, SQL, DNS, or load balancing are examples of IaaS services within the GCP. A client using IaaS has access to all the requested resources and can monitor the entire infrastructure through a website. The user will be paying for resources which are accessed by the user, it has a pay per use concept.

2.) Training with PaaS:

Training the machine learning models through Platform as a Service (PaaS) delivers efficient lifecycle management of machine learning models. It delivers the best of both iterative software development and model management for users. It makes easy for the user to do the configuring and provisioning environments for training and serving machine learning models. Users can call API’s for any complex training jobs. Cloud providers such as Amazon, Google, IBM, and Microsoft have built robust ML PaaS offerings. For example, Microsoft Azure provides training with PaaS which can be used for machine learning and deep learning projects. Users can build, train, and track machine learning and deep learning models in the Microsoft Azure’s workspace.

3.) Training with SaaS

Training the machine learning models through Software as a Service (SaaS) is the most abstract cloud layer. SaaS is typically administered through a subscription model, and it mitigates the need for downloading and installing applications.

SaaS products are available over the web and are highly available for clients. It is delivered as a set of APIs so that workload on the ML algorithms can be reduced. In SaaS, the visibility and control is least compared to other services. The vendor handles all data, middleware, servers, and storage.
