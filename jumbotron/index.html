AWS offers a broad set of machine learning services with supporting cloud infrastructure, putting machine learning in the hands of every developer, data scientist, and expert practitioner.

What is AWS Machine Learning?

Amazon Machine Learning is a service that allows users to develop predictive applications by using algorithms, mathematical models based on the user’s data.

It provides visualization tools to guide the users through the process of creating machine learning (ML) models without having to learn complex ML algorithms and technology. It reads data through Amazon S3, Redshift, and RDS then visualizes the data through the AWS Management Console and the Amazon Machine Learning API.

This data can be imported or exported to other AWS services via S3 buckets.

It uses an “industry-standard logistic regression” algorithm to generate models.

Types of Tasks Performed by Amazon Machine Learning

Three different types of tasks can be performed by Amazon Machine learning service

  • A binary classification model: It can predict one of the two possible results, i.e. either yes or no.
  • A multi-class classification model: It can predict multiple conditions. For example, it can track customer's online orders.
  • A regression model: Regression models can predict the best-selling price for a product or the number of units that will sell.

Features of Amazon Machine Learning

It is easy to create ML models from data stored in Amazon S3, Amazon Redshift, Amazon RDS, and query these models for predictions by using Amazon ML APIs and wizards.

Amazon ML prediction APIs can be used further to generate billions of predictions for the applications. We can use them within the interactive web, mobile, or desktop applications.
